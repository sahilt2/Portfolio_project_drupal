## Portfolio in Drupal

This is a project for Drupal course in Business College Helsinki

### Theme used

- thex
- xara

Both themes are provided by [Drupar](https://drupar.com/)

- Themes Installation
  xara

```shell
composer require 'drupal/xara:^10.1'
```

thex

```shell
composer require 'drupal/thex:^10.0'
```
